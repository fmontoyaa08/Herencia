Mi Tarea de Herencia  - Fátima Montoya
​¡Hola! Este es mi proyecto de Java donde estuve practicando cómo pasar datos de una clase a otra (Herencia) y cómo cambiar lo que hacen los métodos según lo que necesitemos (Override).
​ ¿Qué hay aquí adentro?
​Hice varios ejemplos para que se vea cómo funciona la programación:
​Animales y Perros: El perro hereda de animal pero ladra a su manera. 
​Buses, Motos y Carros: Cómo se pasan la marca y la velocidad entre ellos. 
​Figuras (Círculos y Rectángulos): Cada uno calcula su área con su propia fórmula. 
​Libros: Diferencia entre un libro físico y uno digital (MB). 
​Sistema de Facturas (El reto): * Si es al contado, te aplica un descuento. 
​Si es a crédito, te mete un recargo y te dice de cuánto quedan las cuotas. 
​Y lo más importante: ¡No deja que el total sea negativo! (Validación).
